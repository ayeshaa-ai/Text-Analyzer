#  Text Analyzer using Python

A menu-driven Python program that performs multiple text analysis operations such as character count, word count, vowel count, palindrome checking, and keyword search. This project demonstrates core Python concepts using a simple command-line interface.

---

## Features

- Character count (with and without spaces)
- Word count
-  Vowel count
-  Palindrome check
-  Keyword search with occurrence count and position
-  Menu-driven program using a loop

---

##  Concepts Used

- Functions
- while loop
- if-elif-else
- String methods
- User input handling
